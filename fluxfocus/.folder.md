# Folder: fluxfocus

> Trigger: When this folder's structure/responsibilities/file list changes, update this document.
> 触发条件:当本文件夹的结构/职责/文件列表变化时,更新此文档。

This folder owns the full app UI, shared studio chrome, local state, NFC read/write, App Clip handoff acceptance, Family Controls shielding, the wheel-based session studio with inline shield selection and a custom rounded picker host, and the chain-first home showcase. / 本文件夹负责完整 App UI、共享 studio 视觉层、本地状态、NFC 读写、App Clip 移交接收、Family Controls 屏蔽、带内联屏蔽选择与自定义圆角选择器宿主的滚轮会话控制台，以及链条优先的首页展示。
It consumes hosted invocation URLs, custom scheme handoffs, Screen Time authorization state, and WheelPickerKit UI primitives, then produces persisted sessions, NFC-governed completion and exit decisions, compact tag snapshots, settings-level shield authorization, live shields, derived home-chain visuals, and one consistent studio surface across the full app. / 它消费托管 invocation URL、自定义 scheme handoff、屏幕使用时间授权状态与 WheelPickerKit UI 原语，并产出持久化会话、由 NFC 驱动的完结与退出决策、紧凑标签快照、设置层的屏蔽授权、实时屏蔽、派生首页链条视觉，以及完整 App 统一的 studio 视觉表面。
The invariant is that every physical tag maps to one short `/i/<tagPublicId>` URL as NDEF record 1, one optional snapshot record 2, one minute-level wheel-based duration draft, one inline Focus Shield picker in the session studio, one derived home-chain snapshot, and at most one active session that can only end through the matching tag. / 最重要的不变量是每张物理标签都对应一个作为 NDEF 第 1 条记录的短 `/i/<tagPublicId>` URL、一个可选的第 2 条快照记录、一份分钟级滚轮时长草稿、会话控制台中的一个内联 Focus Shield 选择器、一份派生首页链条快照，并且最多只有一个活跃会话只能通过匹配标签结束。

## Files
- `ContentView.swift`: composition root - tabs, NFC-governed invocation handling, wheel-based session studio with inline shield picker host, and settings UI / 标签页、NFC 驱动的 invocation 处理、带内联屏蔽选择器宿主的滚轮会话控制台与设置界面
- `FocusDomainLogic.swift`: pure contracts - lifecycle planning, manual-exit reason normalization, and tag-touch routing without persistence dependencies / 纯契约层、无持久化依赖的生命周期规划、主动退出原因归一化与标签触碰路由
- `FluxFocusAppStore.swift`: domain service - URL generation, NFC session transitions, precedent rules, and tag snapshots / URL 生成、NFC 会话迁移、判例规则与标签快照
- `FluxFocusModels.swift`: schema - SwiftData entities plus precedent-reason and NFC snapshot value types / SwiftData 实体，以及判例原因与 NFC 快照值类型
- `HomeChainShowcase.swift`: home module - visual-first chain theater, momentum pulses, and chain-node highlights / 首页模块、视觉优先链条剧场、动量脉冲与链节点高亮
- `FocusShieldController.swift`: shield bridge - Family Controls authorization, picker state, ManagedSettings application, and test-safe fallback / Family Controls 授权、选择器状态、ManagedSettings 应用与测试安全回退
- `StudioChrome.swift`: UI primitives - shared studio theme, surfaces, controls, countdown ring, and text-entry styling / 共享 studio 主题、表面、控件、倒计时圆环与输入样式
- `Info.plist`: bundle config - full app metadata and custom handoff URL scheme / 完整 App 元数据与自定义 handoff URL scheme
- `NFCManager.swift`: device service - CoreNFC read/write bridge for dual-record URL plus snapshot tags / 面向 URL 加快照双记录标签的 CoreNFC 读写桥接
- `fluxfocusApp.swift`: app entry - resilient model-container bootstrap, environment wiring, incompatible-store quarantine, and unit-test-safe startup / 应用入口、弹性模型容器启动、环境注入、不兼容 store 隔离与单测安全启动
- `fluxfocus.entitlements`: capabilities - NFC, associated domains, and Family Controls entitlements / NFC、关联域与 Family Controls 权限配置
- `Assets.xcassets/`: assets - app icon and colors / 图标与颜色资源
